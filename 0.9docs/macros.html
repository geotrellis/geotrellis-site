geotrellis-macros

The geotrellis-macro project holds macros for the geotrellis project. Macros must be defined in a separate project, since to perform macro expansion, the compiler needs a macro implementation in executable form.

The geotrellis-macro jar is pulled in as dependency from the core geotrellis jar, so you won’t have to worry about pulling this dependency into your project manually.

Checking for NoData

isNoData isData - These functions are for checking if a data type is NoData or not. See NoData for an explination of what NoData means. Using a macro for this check allows for different overloaded versions of the functions to inline their specific NoData check for the different datatypes; for example,

  val raster:Raster = ???

  val i:Int = raster.get(0,0)
  val d:Double = raster.getDouble(0,0)

  if(isNoData(i)) { println("Is a NoData value.") }
  if(isData(i)) { println("Is a data value.") }
  if(isNoData(d)) { println("Is a NoData value.") }
  if(isData(d)) { println("Is a data value.") }
